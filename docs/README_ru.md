# Formula Copy

Расширение Chrome — копируйте формулы KaTeX как чистый исходный код LaTeX.

[English](../README.md)

## Установка

1. Откройте `chrome://extensions`, включите режим разработчика
2. Загрузите распакованное расширение, выбрав этот каталог
3. Откройте chatgpt.com — иконка станет зелёной при активации

## Использование

- Выделите текст с формулами, **Ctrl+C**, вставьте — получите LaTeX (`$…$` / `$$…$$`)
- Нажмите на иконку, чтобы включить/отключить на любом сайте
- Зелёная иконка = включено на текущем сайте, серая = выключено

## Возможности

- Отдельные формулы, смешанный текст, таблицы HTML поддерживаются
- Текст без формул остаётся без изменений
- Не мешает кнопке копирования блоков кода ChatGPT
- Интерфейс на 8 языках
