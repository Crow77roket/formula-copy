Privacy Policy

Effective date: July 1, 2026

Data Collection

Formula Copy does not collect, store, or transmit any personal information, browsing history, or user data of any kind.

Local Storage

The extension stores two preferences exclusively on the user's device via chrome.storage.local: a list of domain names the user has manually enabled, and the interface language preference. This data never leaves the user's device.

Third Parties

This extension does not integrate with any third-party services, analytics, or tracking frameworks.
