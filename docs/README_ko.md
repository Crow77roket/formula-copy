# Formula Copy

Chrome 확장 — KaTeX로 렌더링된 수학 공식을 깨끗한 LaTeX 소스 코드로 복사합니다.

[English](../README.md)

## 설치

1. `chrome://extensions`에서 개발자 모드 활성화
2. '압축해제된 확장 프로그램 로드'로 이 디렉토리 선택
3. chatgpt.com을 열면 아이콘이 녹색으로 변경됩니다

## 사용법

- 수식이 포함된 텍스트를 선택하고 **Ctrl+C**, 붙여넣으면 LaTeX (`$…$` / `$$…$$`)로 변환
- 도구 모음 아이콘을 클릭하여 모든 사이트에서 활성화/비활성화
- 녹색 아이콘 = 현재 사이트에서 활성화됨, 회색 = 비활성화됨

## 기능

- 단일 수식, 혼합 텍스트+수식, HTML 테이블 지원
- 수식 없는 텍스트는 그대로 유지
- ChatGPT 코드 블록 복사 버튼과 충돌 없음
- 8개 언어 UI 지원

## 테스트
