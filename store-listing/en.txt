Copy KaTeX-rendered math formulas as clean LaTeX source code.

Tired of copying a formula from ChatGPT and getting broken Unicode like "∫₀¹ x² dx = ⅓" instead of "\int_0^1 x^2 \, dx = \frac{1}{3}"? Formula Copy intercepts the copy event on any KaTeX page and replaces the clipboard with the original LaTeX source.

Features
• One-click enable on any website — click the extension icon and toggle it on
• Copies inline formulas as $...$ and display (block) formulas as $$...$$
• HTML tables preserved in clipboard — paste into Obsidian and get proper Markdown tables
• Works on ChatGPT, Claude, and any site using KaTeX or MathJax
• Plain text without formulas is left untouched — zero interference
• Lightweight, no tracking, no data collection

How to use
1. Open a page with KaTeX formulas (e.g. chatgpt.com)
2. Click the Formula Copy icon — it turns green when active
3. Select text containing formulas and press Ctrl+C
4. Paste into Obsidian, Typora, Overleaf, or any LaTeX/Markdown editor

The original motivation was the ChatGPT-to-Obsidian note-taking workflow, but it works anywhere KaTeX lives.
