KaTeX로 렌더링된 수학 공식을 깨끗한 LaTeX 소스 코드로 복사합니다.

ChatGPT에서 수식을 복사할 때 "∫₀¹ x² dx = ⅓"와 같은 깨진 유니코드 대신 "\int_0^1 x^2 \, dx = \frac{1}{3}"를 얻고 싶으신가요? Formula Copy가 복사 이벤트를 가로채 클립보드를 원본 LaTeX 소스로 교체합니다.

주요 기능
• 원클릭으로 모든 웹사이트에서 활성화 — 확장 아이콘을 클릭하여 토글
• 인라인 수식은 $...$로, 디스플레이 수식은 $$...$$로 래핑
• HTML 테이블 구조 유지 — Obsidian에 붙여넣으면 Markdown 테이블로 변환
• ChatGPT, Claude 및 KaTeX 또는 MathJax를 사용하는 모든 사이트에서 작동
• 수식이 없는 일반 텍스트는 그대로 유지 — 간섭 없음
• 가볍고 추적 없음, 데이터 수집 없음

사용 방법
1. KaTeX 수식이 포함된 페이지(예: chatgpt.com) 열기
2. Formula Copy 아이콘 클릭 — 활성화되면 녹색으로 변경
3. 수식이 포함된 텍스트 선택 후 Ctrl+C
4. Obsidian, Typora, Overleaf 등 LaTeX/Markdown 편집기에 붙여넣기

원래 ChatGPT에서 Obsidian으로의 노트 작성 워크플로우를 위해 개발되었습니다.
