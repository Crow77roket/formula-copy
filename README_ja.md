# Formula Copy

Chrome拡張機能 — KaTeXで描画された数式をクリーンなLaTeXソースコードとしてコピーします。

[English](README.md)

## インストール

1. `chrome://extensions` を開き、デベロッパーモードを有効化
2. 「パッケージ化されていない拡張機能を読み込む」でこのディレクトリを選択
3. chatgpt.com を開くとアイコンが緑色に変わります

## 使い方

- 数式を含むテキストを選択して **Ctrl+C**、貼り付けるとLaTeX（`$…$` / `$$…$$`）に
- ツールバーアイコンをクリックして任意のサイトで有効/無効を切り替え
- 緑色のアイコン = 現在のサイトで有効、灰色 = 無効

## 機能

- 単一の数式、テキストと数式の混在、HTMLテーブルに対応
- 数式を含まないテキストはそのまま
- ChatGPTのコードブロックコピーボタンに干渉しません
- 8言語のUIに対応
